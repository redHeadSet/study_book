## 개요

> 공유해주신 문서에 따르면, ECC(Everything Claude Code)에서의 '스킬(Skills)'은 AI 에이전트나 커맨드가 호출하여 실행하는 워크플로우 정의와 도메인 지식을 의미합니다.
>
> 단순히 일회성 명령을 내리는 것이 아니라, AI가 특정 작업을 수행할 때 명확하고 효율적으로 움직일 수 있도록 구조화된 가이드라인과 최적화된 개발 패턴을 담고 있습니다.
>
> 주요 특징과 포함된 내용들을 요약해 드릴게요.
>
> #### 1. 주요 역할 및 포함 내용
>
> 스킬은 주로 언어별 모범 사례, API/데이터베이스 패턴, 테스팅 방법론 등을 다룹니다.
>
> * 코딩 표준 및 패턴: 언어별 모범 사례(Coding Standards)와 백엔드(API, DB, 캐싱), 프런트엔드(React, Next.js)의 최적화된 패턴이 정의되어 있습니다.
> *   보안 및 워크플로우: 코드의 보안 리뷰 체크리스트나 테스트 주도 개발(TDD)을 위한 구체적인 단계가 포함됩니다.
>
>     > TDD 워크플로우 스킬 예시:
>     >
>     > 1. 인터페이스를 먼저 정의
>     > 2. 실패하는 테스트 작성 (RED)
>     > 3. 최소한의 코드 구현 (GREEN)
>     > 4. 리팩토링 (IMPROVE)
>     > 5. 80% 이상 커버리지 확인
>
> #### 2. 지속적 학습 (Continuous Learning v2)
>
> ECC 시스템은 '직관(Instinct) 기반 학습 시스템'을 통해 사용자의 작업 세션에서 자주 쓰이는 패턴을 자동으로 추출하고, 이를 재사용 가능한 '스킬'로 변환(클러스터링)하여 스스로 발전하는 기능을 제공합니다.
>
> #### 3. 로컬에서 직접 스킬 생성 가능
>
> 저장소를 분석하여 맞춤형 스킬을 자동으로 만들어주는 Skill Creator 도구가 내장되어 있습니다. CLI에서 아래 커맨드를 실행하면 Git 히스토리 등을 분석해 새로운 `SKILL.md` 파일을 생성해 줍니다.
>
> * `/skill-create` (현재 저장소 분석)
> * `/skill-create --instincts` (직관도 함께 생성)
>
> 현재 ECC 플러그인에는 총 65개의 스킬이 기본적으로 포함되어 있으며, `skills/` 폴더 내에 각 스킬의 명세가 YAML frontmatter 형식과 함께 작성되어 있어 에이전트들이 필요할 때마다 이를 도구 삼아 작업을 수행하게 됩니다.
