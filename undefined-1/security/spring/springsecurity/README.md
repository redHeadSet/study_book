* Spring Security 는 '인증' & '인가' 에 대한 부분을 Filter 흐름에 따라 처리
  * 인증 : Authentication : 본인이 맞는지
  * 인가 : Authorization : 인증된 사용자에 한해, 해당 리소스에 접근이 가능한 권한이 있는지
* 인증 절차 후 인가 절차를 진행
* 인증과 인가를 위해 Principal, Credential 을 ID, PW 로 사용 → Credential 기반 인증
